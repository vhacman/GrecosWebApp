# Documentazione

## Contenuto

Questa cartella contiene tutta la documentazione del progetto GrecosMenu.

## Struttura

```
Documentazione/
├── CHANGELOG.md          # Storico completo delle versioni
├── QRS/                  # QR code stampabili per il locale
│   ├── grecos-qr-menu-1.pdf    # QR menu principale
│   └── grecos-qr-dolci.pdf     # QR sezione dolci
└── releases/             # Note di release per versione
    ├── README.md
    └── RELEASE_v*.md
```

## File

- **CHANGELOG.md**: Cronologia completa dello sviluppo con tutte le feature, bug fix e miglioramenti per ogni versione.
- **QRS/**: QR code stampabili da affiggere nel locale per consentire ai clienti di accedere al menu digitale.
- **releases/**: Note di rilascio dettagliate per ogni versione del software.

## Manuali

I manuali utente si trovano nella cartella root del progetto:
- `manuale-utente.pdf` - Guida per i clienti
- `manuale-admin.pdf` - Guida per l'amministrazione

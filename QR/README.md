# QRS - QR Code Stampabili

## Descrizione

QR code stampabili da affiggere nel locale per consentire ai clienti di accedere alle diverse sezioni del menu digitale.

## File

| File | Descrizione |
|------|-------------|
| `grecos-qr-menu-1.pdf` | QR code principale - rimanda alla homepage del menu |
| `grecos-qr-dolci.pdf` | QR code dedicato - apre direttamente la sezione dolci (`/menu?cat=dolci`) |

## Utilizzo

1. Stampare i PDF in formato A4 o inferiore
2. Affiggere i QR code nei punti strategici del locale
3. I clienti scansionano il codice con la fotocamera del telefono per accedere al menu

## Note

- I QR code includono il logo Grecos al centro
- Il QR dolci permette un accesso diretto alla sezione dessert, utile per tavoli che hanno già finito la pizza
